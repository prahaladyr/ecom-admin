# Ecom Admin (Next.js)

Admin UI for a single-tenant Ecom backend deployment.

## Setup

1. Install deps:

```bash
npm install
```

2. Create `.env.local`:

```bash
cp .env.local.example .env.local
```

Required env:

- `BACKEND_BASE_URL` (typically `http://localhost:8000`)

3. Run dev server:

```bash
npm run dev
```

Open: `http://localhost:3000`
